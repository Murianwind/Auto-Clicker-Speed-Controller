넷플릭스나 플렉스 웹플레이어에서 인트로, 크렛딧 같은 부분을 자동으로 눌러주는 크롬 확장 프로그램입니다.

1. 기능을 적용할 사이트의 키워드를 등록하고 삭제할 수 있습니다.
2. 클릭할 버튼의 키워드를 등록하고 삭제할 수 있습니다.
3. 넷플렉스에서는 영상 마지막에 크레딧 영상 보기 버튼이 나오면 자동으로 다음화를 재생합니다.
4. 플렉스에서는 영상에 자막이 있을 때와 없을 때 설정된 배속 재생을 지원합니다. 배속은 원하는 데로 설정할 수 있고, 사용자가 임의로 배속을 변경하면 변경된 배속을 유지합니다. OSD에는 변경된 배속이 표시되지만 선택 팝업은 변경된 배속이 표시되지 않습니다.
5. 확장 프로그램을 활성화 또는 비활성화 할 수 있습니다.
6. 브라우저 재시작시 확장 프로그램을 자동으로 활성화하도록 하거나 비활성화를 유지하도록 할 수 있습니다.
7. 설정 팝업 UI를 좀 더 이쁘게 다듬었습니다.
8. 넷플릭스 에피소드 페이지에서 다음화가 무한 반복되는 문제를 수정했습니다.
9. 넷플릭스에서 NSES-UHX 메시지가 있는 페이지가 출력되면 자동으로 뒤로가기를 눌러서 에피소드 페이지로 이동합니다.
10. 넷플릭스에서 동영상의 남은 시간이 30초인데 크레딧 영상 보기 버튼이 나오지 않으면 자동으로 다음화로 넘어갑니다.
11. 플렉스에서는 크레딧 건너뛰기 버튼이 나오면 자동으로 다음 화를 재생합니다.
